# Sistema REINF - Análise de Eventos Fiscais

Sistema para análise de eventos do REINF (4020, 2055, 2010)

## Funcionalidades
- Análise do Evento 4020 - Pagamentos
- Análise do Evento 2055 - Aquisição Produto Rural  
- Análise do Evento 2010 - Tomador de Serviços
- Geração de relatórios Excel
- Validações automáticas

## Como usar
1. Faça upload dos arquivos XML
2. Visualize os dados processados
3. Gere relatórios em Excel
